# ガイド大学 Web アプリ

## フォルダ構成

```
├── index.html
├── map.html
├── main.js
├── README.md
├── image/
│   ├── left.png
│   ├── right.png
│   └── ...
├── csv/
│   └── node_detail.csv
├── glb/
│   ├── art2.glb
│   ├── art3.glb
│   └── ...
└── node/
    ├── art2.csv
    ├── art3.csv
    └── ...
```

## フォルダの説明

| フォルダ | 説明 |
|---------|------|
| **image/** | アプリで使用する画像ファイル |
| **csv/** | CSV データファイル |
| **glb/** | 3D オブジェクト（Blender GLB 形式）ファイル |
| **node/** | ノード情報の CSV ファイル |

## ファイルの説明

| ファイル | 説明 |
|---------|------|
| **node_detail.csv** | ノードの詳細情報が記述されている |
| **\*.glb** | 各棟の 3D オブジェクトファイル |
| **index.html** | 行き先選択 Web ページ |
| **map.html** | アニメーション Web ページ |
| **main.js** | メインの JavaScript ファイル |
