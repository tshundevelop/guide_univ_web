〇フォルダの説明は以下の通りである。
image：使用する画像が全て入っている
csv：csvファイルが入っている
glb：オブジェクトのファイルが入っている

〇ファイルの説明は以下の通りである。
node_detail.csv：ノードの情報が記述されている
～.glb：各棟の3Dオブジェクトファイル
index.html：行き先選択Webページ
map.html：アニメーションWebページ

〇アプリの起動
アプリを起動するには、現在のフォルダ構成でindex.htmlを読み込めば良い。
